    simpsAttr

    Stub for implementation of the @[simps] attribute. #

    This file is currently just a stub that creates a no-operation @[simps] attribute. Without this, all declarations in the mathport output for mathlib3 that use @[simps] fail. With the no-operation attribute, the declarations can succeed, but of course all later proofs that rely on the existence of the automatically generated lemmas will fail.

    Later we will need to port the implementation from mathlib3.
