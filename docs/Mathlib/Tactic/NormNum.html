    partial def Lean.Expr.numeral? (e : Lean.Expr) :
    Option ℕ

    Return true if e is one of the following

    • A nat literal (numeral)
    • Nat.zero
    • Nat.succ x where isNumeral x
    • OfNat.ofNat _ x _ where isNumeral x
