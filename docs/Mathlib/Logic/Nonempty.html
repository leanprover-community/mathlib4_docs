    theorem Nonempty.map {α : Sort u_1} {β : Sort u_2} (f : α → β) :
    Nonempty α → Nonempty β

    Given f : α → β, if α is nonempty then β is also nonempty. Nonempty cannot be a functor, because functor is restricted to Type.
