    inductive List.Perm {α : Type u_1} :
    List α → List α → Prop
    • nil: ∀ {α : Type u_1}, [] ~ []
    • cons: ∀ {α : Type u_1} (x : α) {l₁ l₂ : List α}, l₁ ~ l₂ → x :: l₁ ~ x :: l₂
    • swap: ∀ {α : Type u_1} (x y : α) (l : List α), y :: x :: l ~ x :: y :: l
    • trans: ∀ {α : Type u_1} {l₁ l₂ l₃ : List α}, l₁ ~ l₂ → l₂ ~ l₃ → l₁ ~ l₃

    Perm l₁ l₂ or l₁ ~ l₂ asserts that l₁ and l₂ are Permutations of each other. This is defined by induction using pairwise swaps.
